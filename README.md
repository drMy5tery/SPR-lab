# Speech Processing and Recognition (SPR) Lab Guide

This directory contains the lab programs and assessments for the SPR course. Below is a structured guide to help you navigate through the programs.

## Lab Programs Overview

| Lab | Topic | Description | Key Files |
| :--- | :--- | :--- | :--- |
| **Lab 0** | **Basic Signals Generation** | Generation and plotting of elementary signals (Unit Step, Impulse, Ramp, Exponential, Sinusoidal). | [Lab0](./Lab0) <br> [Notebook](./Lab0/2448540_lab0.ipynb) |
| **Lab 1** | **Sampling & Reconstruction** | Study of sampling theorem, reconstruction (ZOH, Linear Interpolation), and quantization of speech signals. | [Lab1](./Lab1) <br> [Notebook](./Lab1/2448540_Lab1_SPR.ipynb) |
| **Lab 2** | **DTFT & DFT Analysis** | Computing and plotting Discrete Time Fourier Transform (DTFT) and Discrete Fourier Transform (DFT). | [Lab2](./Lab2) <br> [Notebook](./Lab2/2448540_lab2.ipynb) |
| **Lab 3** | **Speech to Text** | building a simple Speech-to-Text application. | [Lab3](./Lab3) <br> [Notebook](./Lab3/2448540_SPR_Lab3_Speech_To_Text.ipynb) |
| **Lab 4** | **LPC & Cepstral Analysis** | Speech signal analysis using Linear Predictive Coding (LPC) and Cepstral Analysis for pitch detection. | [Lab4](./Lab4) <br> [Notebook](./Lab4/2448540_lab4.ipynb) |
| **Lab 5** | **Time Alignment (LTN)** | Linear Time Normalization to align speech sequences of the same word spoken at different speeds. | [Lab5](./Lab5) <br> [Notebook](./Lab5/2448540_lab5.ipynb) |
| **Lab 6** | **Dynamic Time Warping (DTW)** | Implementation of DTW for comparing and aligning two temporal sequences. | [Lab6](./Lab6) <br> [Notebook](./Lab6/2448540_lab6_DTW.ipynb) |
| **Lab 7** | **Spectrogram Analysis** | Visualizing speech signals using spectrograms to analyze frequency changes over time. | [Lab7](./Lab7) <br> [Notebook](./Lab7/2448540_SPR_Lab7.ipynb) |
| **Lab 8** | **Hidden Markov Models (HMM)** | Understanding the components of HMM (Initial, Transition, Emission probabilities) for speech modeling. | [Lab8](./Lab8) <br> [Notebook](./Lab8/2448540_SPR_LAB8.ipynb) |
| **Lab 9** | **Viterbi Algorithm** | Implementing the Viterbi algorithm to find the most likely sequence of hidden states. | [Lab9](./Lab9) <br> [Notebook](./Lab9/2448540_SPR_Lab9.ipynb) |
| **Lab 10** | **Speech Directory System** | Speech recognition-based telephone directory access system using spelled-name recognition. | [Lab10](./Lab10) <br> [Notebook](./Lab10/2448540_SPR_Lab10.ipynb) |
